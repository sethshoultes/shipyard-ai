## Lessons from Great Minds Agency
- Debate: 2 rounds is the sweet spot — enough to align, not wasteful
- Sub-agents on Haiku: ~5x cheaper, sufficient for execution tasks
- QA must run continuously, not just at the end
- Never use in-memory state — always persist to disk/DB
- Feature branches + PRs mandatory — no direct pushes to main
- Token/cost tracking from day one prevents runaway spending
