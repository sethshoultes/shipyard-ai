Near-black background with neon accents. True dark mode for developers who live in the terminal.

Monospace Typography

JetBrains Mono throughout. Every character counts when you're reading code.

High Contrast Colors

Neon green and cyan accents ensure visibility. Semantic colors for all UI states.

Grid-Heavy Layout

Responsive grid system. Clean alignment. Technical precision in every pixel.

Semantic HTML5

    --color-bg: #0d0d0d;

    --font-family: 'JetBrains Mono';

}

}

API Documentation Preview

Quick reference for common properties:

--color-primary

Neon green accent color. Use for primary actions and highlights.

#00ff88

--color-accent

Cyan secondary accent. Use for secondary actions and interactive states.

#00d4ff

--color-bg

Near-black background. The foundation of the dark theme.

#0d0d0d

--font-family

JetBrains Mono monospace font. Default for all text elements.

Palette Two
'JetBrains Mono'

--spacing-4

Standard spacing unit. Base unit for padding and margins.

1rem (16px)

--shadow-md

Medium shadow for depth. Creates visual hierarchy.

0 4px 6px -1px rgba(0,0,0,0.5)
