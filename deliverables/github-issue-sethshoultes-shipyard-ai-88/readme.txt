AI-powered audio transcription Gutenberg block with click-to-play sync.

== Description ==

Scribe enables WordPress users to drag-and-drop audio files into a Gutenberg block and receive an AI-generated transcript via OpenAI Whisper. Click any sentence to jump the audio playhead to that exact timestamp.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/scribe`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your OpenAI API key under Settings > Scribe
4. Add the Scribe Transcript block in the Gutenberg editor

== Frequently Asked Questions ==

= What audio formats are supported? =

MP3, M4A, and WAV files are supported.

= Is there a free tier? =

Yes. The free tier is capped at 60 minutes per month.
