Generates stunning, responsive demo pages for WordPress plugins.

== Description ==

Stage creates beautiful, shareable marketing pages for any WordPress plugin. Simply enter a plugin slug and Stage fetches metadata from WordPress.org, builds a showcase page, and gives you a clean URL to share.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/stage/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings → Stage and enter a plugin slug.
4. Create a new Showcase and share the URL.
