AI-powered visitor assistance, team note tracking, and agreement management for WordPress.

== Description ==

WP Intelligence Suite gives your site three powerful modules in one lightweight plugin:

**LocalGenius** — Instantly answer visitor questions with a searchable FAQ widget that works out of the box. Seed content covers common scenarios so you are never starting from scratch.

**Dash** — Track team notes, tasks, and decisions with a native WordPress post type. Built-in status tracking (Open, In Progress, Resolved) and @mention support keep your team aligned without leaving wp-admin.

**Pinned** — Manage agreements, checklists, and bookmarks. Create reusable agreement templates, attach category taxonomies, and pin frequently referenced docs for quick access.

All three modules ship with seed data on activation, so you see value immediately. Free tier includes unlimited access to core features. Pro tier unlocks extended usage and priority support.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wp-intelligence-suite`.
2. Activate the plugin through the WordPress Plugins menu.
3. Navigate to Settings > Intelligence to configure your license key and preferences.

== Frequently Asked Questions ==

= Does this plugin require an external API? =
No. All features run entirely within WordPress. No external API keys or monthly service fees are required.

= What happens when I activate the plugin? =
Each module creates sample content automatically so you can explore features immediately without manual setup.

= Can I use this on a multisite installation? =
Yes. The plugin works on multisite when network-activated or activated per-site.

= How do I upgrade to Pro? =
Click the Upgrade to Pro button on the Settings > Intelligence page and enter your license key after checkout.

= Will activating or deactivating the plugin delete my content? =
No. Deactivation only clears temporary transients. All your notes, agreements, and FAQs remain safe in the database.
