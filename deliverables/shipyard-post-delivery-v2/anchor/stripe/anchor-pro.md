## Stripe Setup Instructions

### Step 1: Create Product

1. Log into Stripe Dashboard
2. Go to **Products** → **Add Product**
3. Enter:
   - **Name:** Anchor Pro
   - **Description:** Priority site care. 200K tokens, 24-hour response, quarterly refresh proposals.
   - **Image:** (optional) Anchor logo

### Step 2: Create Price

1. In the product, click **Add Price**
2. Enter:
   - **Price:** $149.00
   - **Billing period:** Monthly
   - **Currency:** USD

### Step 3: Generate Payment Link

1. Go to **Payment Links** → **Create Payment Link**
2. Select the Anchor Pro price
3. Settings:
   - **Confirmation page:** Custom (or default)
   - **Collect customer email:** Yes
   - **Allow promotion codes:** Optional
4. Copy the generated URL

### Step 4: Add to Email Templates

Replace `{{MAINTENANCE_LINK}}` in email templates with a landing page showing both tiers, or use direct link to Pro.
