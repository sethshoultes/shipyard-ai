# DREAM Brainstorm: Steve Jobs Perspective
**Date:** 2026-04-23
**Focus:** Design quality, user delight, simplicity

---

## Idea 1: **TypeTide**
*AI typography that makes every post feel like a magazine*

**What:** A WordPress plugin that analyzes post content and applies curated, accessible typography with one click. Not just font swapping — real editorial hierarchy: measure, leading, paragraph indentation, quote styling, caption treatment.

**Why it delights:**
- Select a mood ("Essay," "Tech Doc," "Story") and the entire post transforms instantly
- Live preview in the block editor — no guessing, no reloading
- Dark mode that respects the content, not just inverts colors
- Zero configuration after install; it learns from your theme and improves it

**User:** Publishers, bloggers, indie media, anyone who writes on WordPress and hates ugly defaults
**Distribution:** WordPress plugin directory, design Twitter, indie publication newsletters
**Dogfoodable:** Yes — Shipyard's blog becomes the reference implementation

---

## Idea 2: **EchoFrame**
*Turn any blog post into a shareable audiogram in one click*

**What:** A WordPress plugin that uses our Remotion + TTS pipeline to generate 30-60 second audiogram videos from posts. Not slideshows — editorial motion design with waveform visualization, highlighted pull quotes, and brand-matched color palettes.

**Why it delights:**
- One click from post editor → MP4 ready for social
- Voice that sounds human, not synthesized
- Templates designed like film title sequences, not PowerPoint
- Auto-extracts the best quote and headline; no copy-paste assembly required

**User:** Content marketers, solo founders, newsletter writers who need social assets
**Distribution:** WordPress ecosystem, content creator communities, ProductHunt
**Dogfoodable:** Yes — every Shipyard blog post becomes a social video automatically

---

## Idea 3: **ClipCard**
*Beautiful link previews for the modern web*

**What:** A web tool + WordPress embed that turns any URL into a designer-quality preview card. Paste a link, get a card with clean headline typography, extracted image, and source attribution that doesn't look like it was built in 2012.

**Why it delights:**
- Cards that respect the destination site's brand colors (extracted, not guessed)
- Typography that matches the embed context
- Responsive by default, accessible by default
- Works in newsletters, blog posts, Notion docs, anywhere HTML lives

**User:** Newsletter writers, bloggers, curators, knowledge workers
**Distribution:** Web tool (free), WordPress embed plugin, ProductHunt
**Dogfoodable:** Yes — Shipyard uses it in every roundup post and newsletter
