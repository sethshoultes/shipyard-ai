# Dream Cycle: Steve (Design Quality, User Delight)

Date: 2026-04-25
Phil Jackson, Agency Lead

---

## Idea 1: ScrollTale
**What:** A WordPress plugin that adds "Cinematic Reading Mode" to any long-form post. Readers toggle it via a floating button. The plugin uses the existing H2/H3 structure to create chapter-based navigation with subtle parallax backgrounds, fade-in typography, and progress indicators.

**Why it delights:** WordPress reading has been static since 2003. ScrollTale turns a 3,000-word essay into an experience that feels like a well-designed digital publication — no page builder required, no theme changes, one toggle.

**User:** Content creators who care about reading experience (substack migrants, editorial teams, indie publishers).

**Distribution:** WordPress.org plugin directory; content creators share screenshots because it looks beautiful.

**Build complexity:** Low. IntersectionObserver + CSS transitions + a React toggle. Reads the DOM structure that already exists.

---

## Idea 2: CommandBar
**What:** Raycast for WordPress admin. Hit `CMD+K` (or `Ctrl+K`) anywhere in wp-admin and get a blazing-fast command palette. Search posts, pages, users, plugins, settings. Run actions: "New Post," "Clear Cache," "Toggle Maintenance Mode." Extensible by other plugins via a single action hook.

**Why it delights:** WordPress admin is a maze of nested menus. CommandBar turns navigation from 4 clicks into 2 keystrokes. It makes WP feel like a modern OS, not a 2003 CMS.

**User:** WordPress power users, developers, agencies managing dozens of sites.

**Distribution:** WordPress.org; agency blogs; Twitter demos showing sub-second navigation.

**Build complexity:** Low-Medium. React command palette (cmdk or similar), REST API endpoints for search, keyboard listener mounted in admin header.

---

## Idea 3: FocusWrite
**What:** One-button distraction-free writing for Gutenberg. A zen-mode toggle in the top toolbar. When active, everything fades away except the writing canvas — no sidebars, no top bar, no blocks panel, no breadcrumbs. Just content and a subtle word count. Hit `Escape` to exit.

**Why it delights:** Gutenberg is powerful but noisy. Writers need a single button that says: "Let me think." FocusWrite respects the craft of writing by removing the tool between the writer and the words.

**User:** Bloggers, journalists, newsletter writers who draft in WordPress.

**Distribution:** WordPress.org; writing communities; productivity Twitter.

**Build complexity:** Very low. CSS-heavy, minimal JS. Toggle a body class, hide chrome with CSS, persist preference via user meta.

---

*Steve's pick: CommandBar. It's the one that democratizes delight — not just for readers, but for every person who logs into WordPress every day.*
