## Idea 1: WP AutoSEO
**What:** A WordPress plugin that uses Cloudflare Workers AI to automatically generate meta descriptions, alt text, Open Graph images, structured data (JSON-LD), and internal linking suggestions at publish time. No external API keys needed if you're on Cloudflare. One-click bulk optimization for existing post libraries.

**User:** WP site owners, SEO agencies, content publishers.

**Distribution:** WordPress.org (SEO category is top-10 search volume). AppSumo for agency bulk licenses. SEO newsletter sponsorships.

**Market:** 500M+ WP sites. SEO plugins are a $200M+ category. Rank Math and Yoast prove the market; this undercuts on price (freemium) and outperforms on AI generation speed.

**Technical feasibility:** HIGH. We already integrate Cloudflare Workers AI. The plugin is essentially an async hook on `save_post` + an admin UI. Can ship core version in one session.

## Idea 2: AgentBridge (WordPress MCP Server)
**What:** A WordPress plugin that exposes any WP site as a Model Context Protocol (MCP) server. AI agents can read posts, write drafts, manage ACF fields, query WooCommerce orders, and upload media—without brittle screen scraping or custom API wrappers. Install → copy endpoint → paste into Claude Desktop / Cursor / any MCP client.

**User:** Developers, AI agencies, power users orchestrating multi-agent workflows against live WP sites.

**Distribution:** WordPress.org. Hacker News (MCP is hot). Claude Desktop community forums. GitHub trending.

**Market:** MCP is becoming the USB-C of AI tool integration. WordPress powers 43% of the web. The intersection is essentially empty right now. First-mover advantage in a category that will be enormous.

**Technical feasibility:** HIGH. We already built multi-agent orchestration. MCP is stdio/sse with a JSON schema. A WP plugin can expose an MCP endpoint via REST or a bundled PHP SSE stream. Core tools: `list_posts`, `get_post`, `create_draft`, `update_post_meta`, `upload_media`. One session, clean architecture.

## Idea 3: ClipForge
**What:** A Remotion + TTS API service. Input a blog post URL or RSS feed item; output a vertical (9:16) video with auto-highlighted text, smooth transitions, background footage from Pexels, and TTS narration. Returns MP4 + subtitle file. WordPress plugin auto-generates a video embed alongside published posts.

**User:** Content creators, news sites, affiliate marketers who want YouTube Shorts / TikTok from existing content.

**Distribution:** WordPress.org plugin. Product Hunt. Indie Hackers. Newsletter sponsorships.

**Market:** Short-form video is not optional anymore. 10M+ creators repurpose blog content manually. This automates the pipeline.

**Technical feasibility:** MEDIUM-HIGH. We know Remotion + TTS. The rendering is the bottleneck. Ship as a Cloudflare Worker that queues to a render worker. MVP can pre-render 5 templates. One session gets the core pipeline and WP integration working.
