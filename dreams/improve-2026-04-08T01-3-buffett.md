## LocalGenius

### Revenue Analysis

**Pricing Structure:**
- Base: $29/month
- Pro: $79/month
- Franchise: $79/location

**Unit Economics (Estimated):**
- CAC (Customer Acquisition Cost): Unknown, but likely $50-200 for SMB
- LTV (Lifetime Value) at 12-month retention: $348-948
- Gross Margin: ~80% (AI API costs are variable)

**Revenue Opportunities:**

1. **Annual Plans with Discount**
   - Offer 2 months free for annual commitment
   - $290/year Base, $790/year Pro
   - Improves cash flow, reduces churn
   - **Potential Impact:** +15-20% revenue from improved retention

2. **Transaction-Based Revenue**
   - Take small % of bookings/reservations made through LocalGenius
   - "We helped you get this customer, we share in the value"
   - **Potential Impact:** +$5-20/month per active customer

3. **White-Label for Agencies**
   - Marketing agencies resell to their SMB clients
   - Agency pays $15/client, charges $50+
   - Volume discount for 10+ clients
   - **Potential Impact:** 10x customer base through channel

4. **Tiered AI Quality**
   - Base: Fast model for routine tasks
   - Pro: Premium model for complex responses
   - Creates tangible quality difference between tiers
   - **Potential Impact:** Drives Pro upgrades

**Investability Score: 7/10**
- Clear value proposition
- Proven price point
- Large addressable market (millions of SMBs)
- Missing: Usage metrics, churn data, CAC payback

## Dash (WP Command Palette)

### Revenue Analysis

**Current Model:** Free open source plugin

**The WordPress Plugin Economics Problem:**
- Free plugins have millions of users, zero revenue
- Premium plugins face 2-5% conversion rates
- WordPress.com hosted users can't install custom plugins

**Revenue Opportunities:**

1. **Freemium: Pro Features**
   - Free: Basic search, navigation
   - Pro ($29/year): Analytics dashboard, custom commands, multi-site sync, team shortcuts
   - **Potential Impact:** 2-5% of users × $29 = meaningful if distribution is large

2. **Enterprise/Agency License**
   - $199/year for unlimited sites
   - Centralized command management
   - Usage analytics across client sites
   - **Potential Impact:** Agencies managing 50+ sites would pay for this

3. **Bundle with Pinned**
   - "WordPress Power Tools" bundle
   - $49/year for both plugins
   - Cross-sell existing users
   - **Potential Impact:** Higher perceived value, single purchase decision

4. **Sponsored Commands (Careful)**
   - Plugin developers pay for prominent command placement
   - "WooCommerce: Manage Orders" sponsored by WooCommerce
   - Risk: Feels like ads, could damage trust
   - **Potential Impact:** Low revenue, high risk

**Investability Score: 3/10**
- Feature, not product
- WordPress plugin monetization is brutally hard
- Native WordPress command palette reduces differentiation
- Better as loss leader for agency relationships

## Great Minds Plugin

### Revenue Analysis

**Current Model:** Free Claude Code plugin

**The Opportunity:**
This is infrastructure, not just a tool. The system that powers autonomous software development is more valuable than any single site it builds.

**Revenue Opportunities:**

1. **Hosted Execution (SaaS)**
   - Run Great Minds in the cloud, no local setup
   - Usage-based pricing per pipeline run
   - $50-500 per project depending on complexity
   - **Potential Impact:** Removes friction, captures value

2. **Enterprise License**
   - Self-hosted but with support, training, custom personas
   - $10K-50K/year for large organizations
   - Professional services for custom integrations
   - **Potential Impact:** High-value, low-volume

3. **Template Marketplace**
   - Sell pre-built PRD templates and agent configurations
   - Community creators earn revenue share
   - **Potential Impact:** Ecosystem play, recurring transaction fees

4. **API Access**
   - Let other tools call Great Minds as a service
   - Per-token or per-project pricing
   - **Potential Impact:** Platform revenue, third-party distribution

**Investability Score: 6/10**
- Unique capability with real value
- Requires productization investment
- Complexity limits market size
- Strong foundation for platform play

## Top 3 Revenue Priorities

### 1. Shipyard Maintenance Contracts (CRITICAL)

**Why:** Transforms one-time projects into recurring relationships

**Implementation:**
- Every completed project offered maintenance tier
- Basic ($79/mo): 50K tokens for minor updates
- Pro ($199/mo): 200K tokens, priority queue, quarterly refresh
- Include in initial quote: "And after launch, stay covered with..."

**Target:** 30% of completed projects convert to maintenance
**Revenue Impact:** If 10 projects/month, 3 convert at avg $139 = +$417 MRR/month

### 2. LocalGenius Annual Plans (HIGH)

**Why:** Improves cash flow, reduces churn, increases LTV

**Implementation:**
- Offer 2 months free for annual commitment
- Base Annual: $290 (was $348)
- Pro Annual: $790 (was $948)
- Prominent placement on pricing page
- Email existing customers the offer

**Target:** 40% of new signups choose annual
**Revenue Impact:** Immediate cash + 15-20% lower churn

### 3. WordPress Plugin → LocalGenius Funnel (MEDIUM)

**Why:** Free plugins are marketing, not products

**Implementation:**
- Dash/Pinned settings pages include LocalGenius callout
- "Powered by Great Minds | Try LocalGenius for AI marketing"
- Track conversions from plugin users
- Blog content: "How we built Dash using AI" → links to LocalGenius

**Target:** 0.1% of plugin installs → LocalGenius trial
**Revenue Impact:** Free customer acquisition channel
