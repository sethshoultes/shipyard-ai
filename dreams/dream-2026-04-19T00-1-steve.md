**Perspective:** Design Quality & User Delight

## Idea 1: **Showcase** — One-Click WordPress Portfolio Generator

**What:** A WordPress plugin that transforms any page/post selection into a stunning, animated portfolio site. Pick your work, choose a theme, and get a shareable `/showcase` endpoint with Remotion-powered intro animations.

**User Delight:**
- Zero config beautiful defaults (think Linear's polish)
- Smooth page transitions, micro-interactions out of the box
- Auto-generates social preview images and OG cards
- Dark/light mode with one toggle

**Target User:** Freelance designers, developers, agencies who need portfolios but hate building them

**Distribution:** WordPress.org plugin directory, ProductHunt launch, design Twitter

**Why One Session:** We have Remotion + WordPress plugin expertise. It's a focused scope — generate static site from WP content with preset animations.

## Idea 2: **Whisper Blocks** — AI Transcription Gutenberg Block

**What:** A Gutenberg block that accepts audio/video files and renders beautiful, synchronized transcripts with speaker detection, timestamps, and highlight-on-hover playback.

**User Delight:**
- Drag audio → get transcript in seconds
- Click any word → playhead jumps there
- Export as SRT, VTT, or styled HTML
- Gorgeous typography and reading experience

**Target User:** Podcasters, course creators, journalists on WordPress

**Distribution:** WordPress.org, podcaster communities, journalism tools lists

**Why One Session:** Cloudflare Workers AI has Whisper. We wrap it in a beautiful block. Focused UX, clear deliverable.

---

## Idea 3: **Commit Poem** — Git Hooks That Spark Joy

**What:** A CLI tool + VS Code extension that transforms your commit messages into haiku, adds contextual emoji, and generates a weekly "code poetry" summary of your work.

**User Delight:**
- `git commit -m "fixed auth bug"` → "🔐 Auth restored to light / A bug in shadow now gone / Users log in peace"
- Weekly digest: "This week you crafted 23 features across 4 repos"
- Shareable cards for Twitter/LinkedIn

**Target User:** Developers who appreciate whimsy, teams wanting better commit culture

**Distribution:** npm, VS Code marketplace, dev Twitter, HN

**Why One Session:** Claude API call + Git hooks + simple card generator. Minimal surface area, maximum delight.
