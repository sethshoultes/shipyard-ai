**Focus:** Design quality, user delight, simplicity

## Idea S1: **BlockSmith** — AI Block Pattern Generator for WordPress

**The Pitch:** Every WordPress site looks the same because people use the same tired block patterns. BlockSmith is a WordPress plugin that generates beautiful, unique block patterns from natural language. "A pricing table with a dark theme and glassmorphism" → production-ready blocks in 3 seconds.

**Why It Delights:**
- Zero learning curve — describe what you want, get what you described
- Patterns match your theme automatically (reads your theme.json)
- One-click insert directly into the editor
- Export patterns to share or sell

**User:** WordPress site builders who want custom designs without hiring a developer
**Distribution:** WordPress.org plugin directory, Product Hunt, WP Facebook groups
**Dogfood:** We use it for agency client sites and our own products

**Build Estimate:** 4-6 hours — Claude API + minimal UI in React

## Idea S2: **Loom Killer** — One-Click Video Explanations with AI

**The Pitch:** Recording a Loom is friction. You have to look presentable, find quiet, re-record when you fumble. Instead: paste a URL or text, select a voice and avatar style, get a professional explainer video in 60 seconds. Remotion renders, ElevenLabs speaks, you ship.

**Why It Delights:**
- No camera, no microphone, no awkward pauses
- Consistent brand voice across all videos
- Edit by editing text, not timeline scrubbing
- Perfect for async communication, docs, onboarding

**User:** Developers documenting features, founders doing product updates, support teams
**Distribution:** Direct-to-developer marketing, indie hacker communities, lifetime deal sites
**Dogfood:** We use it for every PRD walkthrough and client update

**Build Estimate:** 6-8 hours — Remotion + TTS + simple web UI on Workers

---

## Idea S3: **Critique** — AI Design Review for WordPress Sites

**The Pitch:** You built the site. It looks "fine." But is it actually good? Critique is a WordPress plugin that screenshots your pages and gives you actionable design feedback using vision AI. Typography, spacing, color contrast, visual hierarchy — like having a design director on retainer.

**Why It Delights:**
- Instant expert feedback without hiring anyone
- Specific, actionable suggestions with visual annotations
- Tracks improvements over time
- Learn design principles by seeing them applied to YOUR site

**User:** Solo developers, small agencies, WordPress DIYers
**Distribution:** WordPress.org, design Twitter, WP communities
**Dogfood:** We run it on every site before client handoff

**Build Estimate:** 4-5 hours — Vision API + WP plugin scaffold we already have
