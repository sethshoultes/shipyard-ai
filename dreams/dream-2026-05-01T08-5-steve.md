# Dream Cycle: Steve — Design Quality & User Delight

**Date:** 2026-05-01
**Persona:** Steve — The craftsman. Believes great design is a moat.

## Idea 1: Zen Mode for Gutenberg
**What:** A WordPress plugin that replaces the Gutenberg editor chrome with a single-column, iA Writer-esque distraction-free writing environment. Focus mode dims everything except the current paragraph. Typewriter scrolling. Premium typography (system font stack, adjusted for reading). Markdown shortcuts that actually work.

**User:** Content creators, bloggers, editorial teams who live in WordPress but dream in Notion/iA Writer.

**Distribution:** WordPress.org plugin repo. Product Hunt for the writer cohort. Newsletter drops to WP content creators.

**Why it delights:** WordPress writing has felt like filling out a tax form since 2018. This brings joy back to the blank page. Ships as a single JS bundle, one admin toggle.

---

## Idea 2: SnapFrame
**What:** A browser extension + WordPress plugin pair. Designers/screenshot any web page; SnapFrame auto-captures responsive breakpoints, wraps them in beautiful device frames (iPhone, MacBook, minimal window), optimizes them, and drops them into the Gutenberg media library as a lazy-loaded WebP block.

**User:** Designers, agencies, product marketers building case studies and portfolio posts.

**Distribution:** Chrome Web Store + WordPress.org. Agency newsletters. "Built for portfolios" positioning.

**Why it delights:** Currently, capturing a responsive screenshot and making it look good in a blog post takes 8 minutes and 4 apps. This does it in 8 seconds and it looks better than hand-crafted.

---

## Idea 3: FlowState Dashboard
**What:** A tiny SaaS widget (embeddable, or standalone) that connects to GitHub and shows a team's daily momentum: who's shipping, PR velocity, AI-generated standup summaries, and a "flow score." Designed like a premium Apple Watch complication—dense, glanceable, gorgeous in dark mode.

**User:** Engineering managers and product leads in small teams (3–20 devs).

**Distribution:** GitHub Marketplace. Hacker News "Show HN." Embedded in Slack as a daily bot.

**Why it delights:** Most dev dashboards are Jira-colored nightmares. This feels like a fitness tracker for your team's creative output. One glance and you know if today is a flow day or a meeting day.
