# DREAM Brainstorm: Steve Jobs Perspective
**Date:** 2026-04-21
**Focus:** Design quality, user delight, simplicity

---

## Idea 1: **Promptfolio**
*A beautiful portfolio generator for AI practitioners*

**What:** A WordPress plugin that generates stunning, Apple-esque portfolio sites showcasing AI prompts, workflows, and case studies. Think "Dribbble meets prompt engineering."

**Why it delights:**
- One-click import from Claude/ChatGPT conversation exports
- Automatic visual hierarchy — prompts become hero cards with elegant typography
- Built-in "Try this prompt" widget that lets visitors experience your work
- Dark mode that actually looks good (not just inverted colors)

**User:** AI consultants, prompt engineers, developers showing off AI integrations
**Distribution:** WordPress plugin directory, ProductHunt launch
**Dogfoodable:** Yes — we'd use it to showcase Shipyard's work

---

## Idea 2: **ClipCraft**
*Turn any blog post into a polished video in 60 seconds*

**What:** A WordPress plugin that uses our Remotion + TTS stack to transform posts into short-form video content. Not a janky slideshow — actual motion design.

**Why it delights:**
- Templates designed like Apple keynotes, not PowerPoint
- Smart content extraction — pulls quotes, stats, key points
- Voice options that don't sound like robots pretending to be human
- One-click export to YouTube Shorts, TikTok, LinkedIn

**User:** Content marketers, solo founders, course creators
**Distribution:** WordPress ecosystem, content marketing communities
**Dogfoodable:** Yes — we'd use it for Shipyard content

---

## Idea 3: **Calm Commit**
*A git commit message generator that brings zen to version control*

**What:** A CLI tool (or VS Code extension) that watches your staged changes, understands the context, and suggests commit messages that are actually useful.

**Why it delights:**
- No more "fix stuff" or "wip" commits
- Learns your team's commit style from repo history
- Suggests breaking commits into logical chunks
- Minimal UI — appears only when you need it, disappears when you don't

**User:** Developers who care about clean git history
**Distribution:** npm/brew, VS Code marketplace, developer Twitter
**Dogfoodable:** Yes — we'd use it every session
