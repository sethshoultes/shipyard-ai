# Peak Dental Care

Your smile deserves the best care. A dental practice website built with [Emdash CMS](https://github.com/emdash-cms/emdash) and deployed to Cloudflare Workers.

![Peak Dental Care](screenshot.png)

**Live site:** [dental.shipyard.company](https://dental.shipyard.company)

## About

A professional dental practice website featuring a dark theme with cyan accents, designed for trust and conversion. Built autonomously by the Shipyard AI pipeline from a PRD.

## Tech Stack

- **CMS:** Emdash (Astro-based)
- **Database:** Cloudflare D1
- **Storage:** Cloudflare R2
- **Runtime:** Cloudflare Workers
- **Template:** Marketing
