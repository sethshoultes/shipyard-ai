AI-powered audio transcription block for WordPress. Drag, drop, transcribe.

== Description ==

Scribe turns audio into beautiful, permanent transcripts directly inside the WordPress block editor.

= Features =

* Drag-and-drop audio upload (MP3, M4A, WAV)
* AI-powered transcription via OpenAI Whisper API
* Synchronized, clickable transcript: click a sentence and the audio playhead jumps to that moment
* Export transcript as styled HTML
* Gorgeous, magazine-grade typography on the frontend

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/scribe/`, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings > Scribe and enter your OpenAI API key.
4. Add a Scribe Transcript block in the editor and upload an audio file.

== Frequently Asked Questions ==

= What audio formats are supported? =

MP3, M4A, and WAV files are supported.

= Is there a file size limit? =

The default maximum upload size is 50 MB.

= Where is the transcript stored? =

Transcript data is stored as post meta in your WordPress database.
