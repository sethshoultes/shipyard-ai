# PRD: Emdash Theme Collection — 5 Unique Themes

## Overview
Build 5 visually distinct Emdash themes. Each theme should have its own layout, color palette, typography, and personality. No two themes should look alike. These will be showcased as portfolio pieces and offered to clients.

## Problem
All current Shipyard example sites use the same layout — same hero structure, same card grids, same nav pattern. Clients need to see variety. A theme marketplace needs diversity.

## Themes to Build

### 1. Ember — Bold and Editorial
- Magazine-style layout with large featured images
- Serif headings (Playfair Display or similar), bold color blocking
- Dark navy + burnt orange palette
- Asymmetric grid layout, pull quotes, editorial feel
- Best for: restaurants, creative studios, magazines

### 2. Drift — Minimal and Airy
- Lots of whitespace, thin sans-serif (Inter Light or similar)
- Single accent color (sage green) on white
- Horizontal scroll sections, subtle parallax
- No borders, no cards — content floats freely
- Best for: spas, yoga studios, wellness brands

### 3. Forge — Industrial and Technical
- Dark mode, monospace typography (JetBrains Mono)
- Terminal-inspired UI elements, code-block styling
- Neon green accents on near-black (#0a0a0a)
- Grid-heavy, data-dense, dashboard feel
- Best for: SaaS, developer tools, tech companies

### 4. Bloom — Warm and Organic
- Rounded corners everywhere, hand-drawn illustration style
- Warm cream background (#fef7ed), terracotta accents
- Playful typography (Quicksand or similar)
- Stacked sections with organic shapes, no sharp edges
- Best for: bakeries, florists, childrens brands, local shops

### 5. Slate — Corporate and Clean
- Professional gray palette with blue accents
- System font stack, tight spacing, dense information
- Sidebar navigation, breadcrumbs, structured hierarchy
- Tables, charts, team grids, testimonial carousels
- Best for: law firms, consulting, finance, enterprise

## Requirements per Theme
- Full Emdash template (homepage, about, services, contact, blog)
- Responsive (mobile + desktop)
- Dark mode toggle where appropriate
- Valid HTML/CSS, accessible (WCAG 2.1 AA)
- Self-contained (no external dependencies)
- Screenshot of each page for portfolio

## Also: Redesign Existing Example Sites
The current example sites (Sunrise Yoga, dental, craft, bellas) all use the same layout. Assign each a different theme:
- Sunrise Yoga Studio — use Drift theme (minimal, airy, wellness)
- Bellas Restaurant — use Ember theme (bold, editorial, food photography)
- Bright Smile Dental — use Slate theme (professional, clean, trustworthy)
- Craft Brewery — use Forge theme (industrial, dark, technical)

## Success Metrics
- Each theme visually distinct at a glance
- No shared layout patterns between themes
- Each renders correctly at 375px and 1440px
