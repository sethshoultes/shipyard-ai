ROLLBACK PROCEDURES
================================================================================

If Cron Job Breaks:
  1. Stop calling POST /api/pulse/cron/generate-notifications
  2. Revert to previous git commit
  3. Use manual trigger endpoint: POST /api/pulse/admin/generate [requires API key]

If Database Migration Fails:
  1. Drop new tables: notifications, journal_entries, achievements, etc.
  2. Restore from backup (pre-migration snapshot)
  3. Re-run migration with fixes

If SMS Delivery Fails:
  1. Check Twilio account status + API credentials
  2. Review error logs for spam trigger words
  3. Whitelist sender number if carrier filtering blocking

If Email Delivery Fails:
  1. Check SendGrid API key + domain verification
  2. Review bounce logs for invalid emails
  3. Check spam folder (may be legitimate delivery issue)
