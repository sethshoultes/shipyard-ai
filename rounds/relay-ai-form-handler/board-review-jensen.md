Verdict: Thin API wrapper dressed as WordPress plugin. Clean execution, zero strategic defensibility.

- Moat: None. Hash cache in wp_options is not moat. Per-site data silo kills network effects.
- What compounds: Nothing. Each install is island. Classification accuracy flat with scale.
- AI leverage: 1.2x, not 10x. Haiku tags intent. AI unused for routing logic, reply drafting, anomaly detection, predictive scoring.
- Missing unfair advantage: Collective intelligence across installs. Should train custom classifier on aggregated, human-corrected labels. Best spam/sales detector on internet via federated learning from thousands of sites.
- Missing platform play: No SaaS control plane. Agency with 50 sites juggles 50 isolated inboxes. Need central dashboard, cross-site analytics, vertical benchmarks.
- Platform path: Multi-tenant relay.shipyard.company with OAuth into WP. Fine-tuned model. App store for CRM/Slack/Salesforce integrations. Revenue share with agencies.
- Score: 5/10. Execution is solid. Strategy is a feature, not company. Shippable plugin, not defensible business.
