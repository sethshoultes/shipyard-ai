# Stage — 2-Minute Demo Script

NARRATOR: You wrote twelve thousand lines of PHP. Solved edge cases that don't have names. And your marketing page looks like a parking ticket.

[SCREEN: WordPress.org plugin directory — cluttered screenshot, beige tones]

NARRATOR: That can't be the first impression. Not anymore.

[SCREEN: WordPress admin dashboard. Cursor flies to Settings → Stage]

NARRATOR: Settings. Stage. Type the slug — "my-plugin" — hit save. That was configuration. We're done.

[SCREEN: Settings page. Slug field filled. Submit button clicked. Success message flashes.]

NARRATOR: Now go to /stage/my-plugin/.

[SCREEN: Browser address bar. URL types out. Page loads in one breath.]

NARRATOR: Watch it breathe.

[SCREEN: Hero section resolves. Sharp typography. Gradient shifts subtly. Dark mode follows the system. No beige. No admin blue. No JavaScript bloat.]

NARRATOR: Name. Author. Version. Live metadata from WordPress.org. Cached for twenty-four hours. Zero maintenance. You didn't touch a theme file. You didn't hire a designer.

[SCREEN: Cursor hovers "Install on my site" button. Clean CSS transition.]

NARRATOR: One button. Visitor becomes installer.

[SCREEN: URL copied. Pasted into Twitter/Slack. Rich OpenGraph preview unfurls — image, title, description.]

NARRATOR: Share it anywhere. The social card sells before the click.

[SCREEN: Mobile viewport. Same page, dark mode, flawless typography.]

NARRATOR: Twenty minutes ago you had code and a prayer. Now you have a stage.

[SCREEN: Lockup. Clean URL: /stage/]

NARRATOR: Build great plugins. We'll handle the first impression.
