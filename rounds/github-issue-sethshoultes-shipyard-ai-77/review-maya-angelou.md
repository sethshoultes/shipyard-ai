# Copy Review — Stage

## Verdict

Mostly functional. Little soul. Reads like a manual written by someone who never stood in front of a crowd.

## What Works

- "Install on my site" — direct, personal, breath of fresh air.
- "Stage" as a name — short, theatrical, promising.

## What Drains the Life

- **"Generates stunning, responsive demo pages"** — "stunning" is wallpaper language. Worn thin by every theme shop on earth. No heartbeat.
- **"Stage creates beautiful, shareable marketing pages"** — Three adjectives in a row, none felt. "Marketing pages" is where warmth goes to die.
- **"Plugin Slug"** — Exposed plumbing. You handed the user a wrench and called it a gift.

## Rhythm Check

Sentences march. No swing. No pause. "Simply enter a plugin slug and Stage fetches metadata... builds a showcase page... and gives you..." — list-makers don't seduce.

## Headlines

None stop the scroll. "Stage" is the only one with teeth, and it is the product name, not the copy.

## Trying Too Hard

"Stunning." "Beautiful." "Shareable." Adjectives stacked like cheap jewelry. Let the action speak.

## Rewrites

**Weakest line 1** — Plugin header / readme short description
*Current:* `Generates stunning, responsive demo pages for WordPress plugins.`
*Better:* `Give your plugin a stage worth standing on.`

**Weakest line 2** — readme description opening
*Current:* `Stage creates beautiful, shareable marketing pages for any WordPress plugin.`
*Better:* `Built for the plugin you poured yourself into. One name. One page. Ready to share.`

**Weakest line 3** — Settings label / meta box title
*Current:* `Plugin Slug`
*Better:* `Which plugin are we spotlighting?`
