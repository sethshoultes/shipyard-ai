# Essence

**What is this?**
Markdown files become a publishing system that disappears.

**The feeling:**
Calm. Permanent. Respectful of time.

**One thing that must be perfect:**
Typography and readability.

**Creative direction:**
Invisible infrastructure. Visible craft.

---

**Core tensions resolved:**

- Speed vs. craft → Ship tight, make it beautiful
- Writer vs. reader → Optimize for 10,000 reads, not 1 write
- Now vs. later → Boring tech, radical simplicity
- Vibes vs. metrics → Both. Details are product.

**Non-negotiables:**
- Markdown only (constraint breeds clarity)
- No comments, tags, search in V1
- RSS feed (free distribution)
- Published flag (trust but verify AI content)

**What we're really building:**
A printing press so simple the writer forgets it exists, so readable the audience remembers it.
