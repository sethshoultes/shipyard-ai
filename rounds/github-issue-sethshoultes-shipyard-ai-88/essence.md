# Essence

- Product: WordPress plugin that captures fleeting voice and renders it as beautiful, permanent text.
- Feeling: Reverence. Words matter. Your voice deserves to last.
- Must be perfect: The reading experience — typography, breathing whitespace, the feeling of paging through something crafted.
- Creative direction: Invisible magic. Beautiful permanence.

---

Strip everything that does not serve the words.
No config before wonder.
No feature that does not spark joy.
No utilitarian export that undermines the canvas.

The transcript is not a data dump.
It is a resurrection.
Build that.
