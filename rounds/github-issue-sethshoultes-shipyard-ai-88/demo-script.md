# Scribe — 2-Minute Demo Script

**NARRATOR**
It’s eleven p.m. Your editor wants the story by morning. You’ve got forty-seven minutes of audio and a blinking cursor that hates you.

[SCREEN: Exhausted journalist at kitchen table. Audio waveform frozen on laptop. Notebook covered in scribbles.]

**NARRATOR**
You could send it to a service. Wait three days. Pay a ransom. Or—

[SCREEN: WordPress Gutenberg editor. Fresh post. Cursor blinks in empty block canvas.]

**NARRATOR**
—you drag the file straight into the post.

[SCREEN: Hand drags MP3 file from desktop into the block editor. Drop zone glows. File releases. Upload bar fills.]

**NARRATOR**
That’s it. No wizard. No configuration purgatory. The block takes it, queues the job, and lets you breathe.

[SCREEN: Job status chip cycles: uploading → queued → processing. Zen spinner. No anxiety.]

**NARRATOR**
While you make coffee, Whisper is doing the listening.

[SCREEN: Status flips to "done." Transcript materializes in the editor: clean paragraphs, generous line-height, warm spacing.]

**NARRATOR**
And there it is. Not a wall of text. Magazine-grade. Permanent. Every sentence stamped with its moment in time.

[SCREEN: Cursor clicks a sentence mid-paragraph. Audio player in block scrubs to that exact second and plays.]

**NARRATOR**
Click any sentence. The playhead snaps to it. Click a word—same thing. Because a transcript shouldn’t be a graveyard. It should be a map.

[SCREEN: Frontend view on mobile. Reader scrolls transcript, taps a quote, audio plays from that point. Typography breathes.]

**NARRATOR**
On the front end, it’s gorgeous. System fonts, generous spacing, responsive. No bloat. No watermark screaming across your brand.

[SCREEN: Settings page. One field. API key. That’s the whole settings page.]

**NARRATOR**
Settings? One field. The API key. Paste it. Or define it in wp-config and never see a settings page again.

[SCREEN: Back to editor. Journalist closes laptop, satisfied. Sunrise through window.]

**NARRATOR**
Forty-seven minutes of audio. Transcribed. Synced. Published. While you were still awake enough to enjoy it.

[SCREEN: WordPress post live on site. Transcript beneath embedded audio. Reader clicks, listens.]

**NARRATOR**
Scribe. Because your story’s already there. You just need it written down.

[SCREEN: Scribe logo lockup. "Install from WordPress.org."]

---

*Runtime: approximately 2 minutes.*
