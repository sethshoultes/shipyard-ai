# PAID-TEMPLATE — 2-Minute Demo Script

---

**NARRATOR**
Three weeks ago my cofounder said: "We need a landing page by Friday."

[SCREEN: Calendar app. Friday. Red dot.]

**NARRATOR**
I spent four hours on a brief. Another three in Notion. Then the agency quoted six.

[SCREEN: Cursor blinking in a blank doc. The word "Scope" appears, then three paragraphs of legal boilerplate.]

**NARRATOR**
I don't have six weeks. I have this afternoon.

[SCREEN: Black. Shipyard logo. Clean.]

**NARRATOR**
So I opened Shipyard. Five questions. That's it.

[SCREEN: Intake form. Question 1: "What's your business and who is it for?" User types: "SaaS analytics for dentists."]

**NARRATOR**
Not fifty. Not a spreadsheet. Five.

[SCREEN: Question 2: "What feeling should the site give?" Dropdown: "Trusted. Fast. Human." User selects "Trusted."]

**NARRATOR**
The form doesn't interrogate you. It listens.

[SCREEN: Question 3–5 fly by. "Any sites you love?" "Deadline?" "Pages you need?" User clicks Submit.]

**NARRATOR**
Then it writes the contract. Twenty lines. No commercial fields. No pricing tables buried in a PRD like a landmine.

[SCREEN: A markdown file. Exactly 20 lines. Clean headings: Brand Voice, Pages, Assets, Guardrails.]

**NARRATOR**
And then — the agents wake up.

[SCREEN: Terminal. Three shards spin up. Design → Component → Deploy. Token burn meter in the corner.]

**NARRATOR**
Design figures out the layout. Component builds the pieces. Deploy makes it real. If one hits a wall, it retries. If it fails twice, a human gets a ping. I'm not babysitting.

[SCREEN: Progress bars. Fast cuts. Code blocks resolving into rendered buttons, hero sections, nav bars.]

**NARRATOR**
Eighteen minutes.

[SCREEN: The terminal stops. Silence.]

**NARRATOR**
A URL.

[SCREEN: Browser window. Live site loads. Clean hero. "Analytics your patients trust." CTA button. Mobile responsive. Subtle meta tag flickers in the inspector: "Shipyard."]

**NARRATOR**
I sent it to my cofounder. He replied with one word: "Ship."

[SCREEN: Deploy preview. Two buttons: Approve / Reject. Cursor hovers. Clicks Approve. URL switches from preview to live domain.]

**NARRATOR**
No revision loop. No Jira ticket. No 9 AM standup about button radius.

[SCREEN: Black. Text appears: "Friday came early."]

**NARRATOR**
Shipyard. Built the thing. Moved on.

[SCREEN: Logo. Tagline: "Site by Monday. Or earlier."]

---

*End.*
