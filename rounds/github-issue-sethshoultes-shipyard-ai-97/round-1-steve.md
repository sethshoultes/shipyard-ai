# TRUE

## Product Naming

The name is **TRUE**. One word. Four letters. It is not a linter, not a gate, not a pipeline step. It is the truth your build refuses to tell you. Next.js smiles and says "success" while it ships you a corpse. TRUE says **no**. When TRUE is done, you either have a thing that works or you have nothing. That is what truth looks like.

## Design Philosophy

A smoke test that smokes is already too late. TRUE sits between build and deploy like a flight director at Cape Canaveral. Binary. Red or green. Stop or go. There is no yellow. Yellow is where fear lives — the coward's color for "maybe someone will notice." We do not traffic in maybe. If your API route is declared in source but will vaporize in static export, TRUE kills the launch. Instantly. One sentence. No stack trace. No committee.

## The First 30 Seconds

You add TRUE. You run your build. If everything is actually real — if every declared route can actually breathe in your deployment target — TRUE prints one word: **Ready.** That is all. No badges, no metrics dashboard, no confetti. But if you have a phantom route, a Margaret-class lie hiding in your `out/` directory, the build stops cold and TRUE speaks: *"Your Edge route cannot fly in static export. Move it or lose it."* One breath. You fix it. You are better for it.

## Brand Voice

TRUE speaks like the engineer who saved Apollo 11. Direct. Uncompromising. Respectful of your intelligence, allergic to your excuses. It never says "we detected an issue." It says **"This route is dead on arrival."** It never apologizes for stopping you. It is doing its job, which is to make sure you do yours.

## What We Say NO To

**NO auto-fix.** Magic that moves your code for you breeds ignorance. If you do not understand that static export strips server routes, you do not deserve to ship. TRUE stops you. You learn. Then you ship.
**NO warning levels.** Warnings are lies told by people too afraid to take a stand. Ship or do not ship.
**NO configuration.** If TRUE needs a config file, we failed. It reads your `next.config.ts`. It reads your routes. It knows the truth or it is broken.
**NO noise.** When everything is correct, silence is the only appropriate sound.

## The Emotional Hook

Every developer has felt it — that 2am dread, the Slack message from a customer, the CEO asking why the form submits to a 404. TRUE gives you the courage to deploy on a Friday at 5pm because **you know your parachute was packed by Margaret herself.** You do not hope. You know. And that feeling — that bone-deep confidence — is insanely great.
