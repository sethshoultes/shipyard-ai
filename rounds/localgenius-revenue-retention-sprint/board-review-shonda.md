# Board Review — Shonda Rhimes

## Verdict: Pass on mechanics. Fail on story.

- **Score: 4/10.** This sprint optimizes a cash register, not a relationship. Pretty wrapping paper around a billing toggle.

## Story Arc
- No arc. Signup → pricing fork → confirmation email. That's it.
- "Breathe annually" vs "Go month by month" — decent emotional fork. But no chapter two.
- Missing: onboarding beat, first-win moment, rising action. User pays, then... silence till Sunday.
- Dashboard "time saved" is nice-to-have. PRD buries it. Should've been must-have.

## Retention Hooks
- Weekly digest, Sunday 6 PM. Smart restaurant rhythm.
- But PRD promises *month-over-month* narrative. Code delivers *week-over-week*. Weak.
- One hero metric + time saved. Fine. Not sticky.
- No streaks. No milestones. No "Your 30-day review." No ritual.
- Annual billing is a *lock-in*, not a hook. Don't confuse commitment with desire.

## Content Flywheel
- None. Zero.
- No blog, no community, no user stories, no "feature your restaurant."
- Digest is a receipt, not a magazine. Receipts get archived. Magazines get opened.
- Where's the flywheel? Emails go out. Nothing comes back. Dead end.

## Emotional Cliffhangers
- Subject line: "Sous — Reviews handled this week." Yawn.
- No "Here's what's cooking next week." No preview. No悬念.
- Confirmation email: "You're all set for 12 months." Translation: go away for a year.
- No narrative tension. No "Will next week be better?" No curiosity gap.

## What I'd Greenlight
- Brand voice "Sous" + banned phrases list. That has teeth.
- "Breathe annually" framing. Someone in that room understands emotional choice.
- Digest timing (Sunday 6 PM). Respects the audience's life.

## What Gets a Rewrite
- MoM narrative shipped as WoW. Promise broken.
- Confirmation email kills momentum. Should tease Week 1 preview.
- Missing: "Your next review response is already drafted" — a reason to log in *tomorrow*.
- Need a content loop. User posts → digest features user → user feels seen → user posts more.
- Annual plan needs a midpoint celebration. Month 6: "Halfway. You've saved 40 hours."

## One-Line Fix
> Turn the weekly digest into a *serial* — last week's win, this week's preview, one question that demands an answer. Make them need to open it.
