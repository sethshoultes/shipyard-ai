# Still — Demo Script (2:00)

---

**NARRATOR**
Three a.m. You promised the team you'd ship by morning.

[SCREEN: IDE with a messy diff. Fifty files. Three hours of frantic coding. Cursor blinking in an empty commit message box.]

**NARRATOR**
And now... this. The blank box of shame.

[SCREEN: Typing. Backspacing. Typing again. "fix stuff" — delete. "WIP" — delete. Nothing fits.]

**NARRATOR**
Every message sounds like a confession or a lie. There's a whole language for this, and you forgot how to speak it.

[SCREEN: Terminal window opens. One line typed: `npm install -g still && still install`]

**NARRATOR**
So you install Still.

[SCREEN: Hook installed. Cut back to IDE. `git commit` triggered.]

**NARRATOR**
That's it. No config file. No dashboard. No emoji slider.

[SCREEN: Git diff --staged scrolls past. Still reads it. A suggestion appears in the commit buffer: `Refactor authentication middleware to validate tokens before route handling`]

**NARRATOR**
It reads what you actually did. And writes what you meant.

[SCREEN: Same diff. Second commit attempt. The suggestion appears instantly — sub-second, cached.]

**NARRATOR**
Cache hits under ten milliseconds. Because repeating yourself is the real waste of time.

[SCREEN: Commit history. Clean lines. Imperative mood. No exclamation points. No crying face. Just craft.]

**NARRATOR**
Master craftsman voice. Not marketing. Not panic. The tone of someone who knows their job.

[SCREEN: Split screen. Left: old commit history — "fixed bug!!!", "asdf", "Merge branch 'please-kill-me'". Right: new history — declarative, precise, calm.]

**NARRATOR**
Look at that. You didn't just write better commits.

[SCREEN: Close-up on a single perfect commit message in the log.]

**NARRATOR**
You became the kind of engineer who writes them.

[SCREEN: Fade to black. `still` — calm, useful commit messages from staged changes.]
