# Essence

- **Product:** A WordPress plugin that makes your site listen to AI. You speak. It obeys.
- **Feeling:** Relief. The end of admin fatigue. The moment the future arrives early.
- **Must be perfect:** Thirty-second install. One card. One URL. One token. One copy button. Paste. Done. Forty-five seconds is failure.
- **Direction:** Invisible by design.

---

## Sacred Rules

- No wizards. No toggles. No jargon in UI.
- One white card. One button. Absolute clarity.
- "Your site is now listening."
- If the user notices the interface, we failed.
