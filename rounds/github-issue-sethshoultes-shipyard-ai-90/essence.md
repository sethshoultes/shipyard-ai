# Essence

## What This Really Is
Agency without employees — one human conducting a living team of AI minds.

## The Feeling
Effortless power. You snap your fingers. Work happens while you go to dinner.

## The One Thing That Must Be Perfect
First thirty seconds. Land in a running orchestra. Touch a node. Feel control.

## Creative Direction
Living orchestration. Liquid light. Morning sun. No tooltips. No fear.

## What Gets Cut
- Dark mode toggle. One aesthetic. Defend it.
- JSON-first thinking. Visual canvas *is* the product. JSON hides behind API for devs.
- Empty states. Login walls. Sad faces. No waiting to feel power.
- Drag-and-drop Figma clone. Nodes breathe. Lines pulse. Agents flock.
- Workers AI hand-waving. This runs on Claude. Own it.
- Freemium billing in v1. Stripe burns tokens. Ship value first.

## What Ships Day One
- Visual conductor's baton. Child-usable. Zero docs needed.
- Parallel node execution. Aggressive caching. Latency kills magic.
- Per-user token budgets. Request deduplication. Economics must sustain the feeling.
- One name. One word. Lowercase. Alive. AgentForge is dead.
