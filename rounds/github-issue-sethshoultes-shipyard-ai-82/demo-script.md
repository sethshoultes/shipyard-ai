# Cut — Demo Script (2:00)

---

NARRATOR
You want to know why users don't read changelogs? Changelogs are where good news goes to die. Three months of your life. Four all-nighters. That refactor that nearly ended you. And you bury it in a text file. Bullets. Dashes. Version numbers floating in space like gravestones.

[SCREEN: Black. Raw terminal text blinks into view: "== Changelog =="]

NARRATOR
I shipped a memory-leak fix last Tuesday. Critical patch. Zero applause. Know why? Because I announced it like a parking ticket.

[SCREEN: WordPress admin. Cursor hovers over Tools menu, clicks "Cut"]

NARRATOR
Then I found this. One page. One textarea. I paste the same readme.txt I've been ignoring since 2019. I click Preview.

[SCREEN: Admin page. Raw changelog text fills the textarea. Mouse clicks the "Preview" button.]

NARRATOR
And the screen breathes.

[SCREEN: Full black. White typography fades in. "Version 3.2.0" slides up with calm confidence.]

NARRATOR
No bounce. No wobble. Dignity. Restraint. The kind of motion that makes you trust the person who built it.

[SCREEN: Bullet points emerge sequentially beneath the version. Gentle slide transitions.]

NARRATOR
Then the voice starts. Not some robot from a 2008 GPS. The browser's own voice. Calm. Measured. "Fixed memory leak in query builder. Improved database indexing."

[SCREEN: Web Speech API indicator pulses gently. Each word highlights as it is spoken.]

NARRATOR
My changelog. My work. Read aloud with the gravity it actually deserves.

[SCREEN: WordPress post editor. Shortcode `[cut changelog="..."]` pasted into a block. Cursor clicks Publish.]

NARRATOR
I publish it with a shortcode. One line. No server rendering. No ffmpeg bill. No Lambda function holding my wallet hostage. Just CSS, a little canvas, and the truth.

[SCREEN: Front-end blog post. Animated changelog plays. A visitor stops scrolling, leans in, watches.]

NARRATOR
And for the first time, someone leans in. They don't scroll past. They watch. They listen.

[SCREEN: Final version number holds full screen. Text: "Version 3.2.0"]

NARRATOR
Three months of your life. Finally announced like it mattered. Because it did.

[SCREEN: Fade to black. Logo: Cut. Tagline: "Changelog Theatre."]
