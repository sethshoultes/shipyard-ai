# Pulse — 2-Minute Demo Script

NARRATOR
You swapped the engine. Now you're holding your breath.
[SCREEN: Black. A terminal cursor blinks.]

NARRATOR
Every pipeline promises it works. Most lie quietly.
[SCREEN: GitHub Actions dashboard. Red jobs. Unknown model tags.]

NARRATOR
You need one honest signal. One heartbeat.
[SCREEN: Empty folder. New file: run.sh.]

NARRATOR
One line. No imports. No config. No room to hide.
[SCREEN: Typing: `echo "Kimi drove this." > pulse.txt`]

NARRATOR
Run it.
[SCREEN: Terminal. `./run.sh`. Exit code 0.]

NARRATOR
Again. Prove it's clean.
[SCREEN: Second run. `cat pulse.txt`. One line. No duplication.]

NARRATOR
Now chain it. Push. CI picks it up.
[SCREEN: Workflow triggers. Checkout. Execute.]

NARRATOR
Five seconds. Not a meeting. Five seconds.
[SCREEN: Timer stops at 00:04.12.]

NARRATOR
And the artifact?
[SCREEN: `cat pulse.txt`]

NARRATOR
"Kimi drove this."
[SCREEN: Text fills frame. Green checkmark pulses.]

NARRATOR
That's not a log. That's a pulse. Seth breathes again.
[SCREEN: Fade to black. Badge: Alive.]
