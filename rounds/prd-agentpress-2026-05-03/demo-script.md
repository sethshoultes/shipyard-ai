# AgentPress — 2-Minute Demo Script

---

NARRATOR
Eleven fourteen. Your writer has the flu. Launch is in nine hours and the client just added—don't they always—a dozen posts and hero images to the scope. You're staring at WordPress like it owes you money.

[SCREEN: Dark room. Laptop glow on tired face. Empty WordPress "Add New Post" screen, cursor blinking.]

NARRATOR
You don't need more coffee. You need an agent.

[SCREEN: Quick zip upload. Activate plugin. Tools → AgentPress. Paste API key. Save.]

NARRATOR
Upload. Activate. One key. Done. Now pay attention.

[SCREEN: Terminal window. Hands type: curl -X POST .../wp-json/agentpress/v1/run -d '{"task":"Write 500 words on sustainable packaging, professional tone"}']

NARRATOR
No specs. No tickets. No Jira. One sentence in plain English. Hit return.

[SCREEN: JSON response expands. Cut to WordPress editor — user pastes text, hits Publish. "Sustainable Packaging: A Professional Overview" goes live.]

NARRATOR
Router sees "write." ContentWriter wakes up. You drop the text in and it's live before you finish exhaling.

[SCREEN: Another curl. Task: "Hero image, minimalist, green leaves, 1024 square."]

NARRATOR
Now you need a visual. Same endpoint. Same sentence.

[SCREEN: JSON response with HTTPS image URL. Cut to browser — image loads crisp and clean.]

NARRATOR
ImageGenerator. No dropdown. No routing rules. The plugin just... knew.

[SCREEN: Admin log viewer. Table populates: Time, Capability, Status, Latency. 1.2s. 1.4s. Green checks.]

NARRATOR
Every move logged. Every route visible. Under two seconds, every time.

[SCREEN: Montage: three more posts publish. Two more images load. Clock on screen advances: 11:14 to 11:47.]

NARRATOR
Eight posts. Four images. Thirty-three minutes. And you never once told it which button to push.

[SCREEN: Morning light. Same laptop. Client email: "This is exactly what we wanted." User closes lid, smiles.]

NARRATOR
You didn't learn a platform. You didn't read a manual. You asked. And WordPress, for the first time in twenty years, answered back.

[SCREEN: Black. Logo: AgentPress. Tagline: "WordPress that listens."]

NARRATOR
AgentPress.

---

*Runtime: approximately 2 minutes.*
